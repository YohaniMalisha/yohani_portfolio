Online Movie Rental System

The Movie Rental System is an online platform allowing users to browse, rent, and return movies, with user data and rental history stored in XML files. The application provides a secure and interactive experience, featuring user registration, login, and dynamic updates without page reloads.

Key Features:

  • User Registration & Login: Secure user signup and login with authentication.
  • Browse Movies: View available movies with details like title, genre, and rating.
  • Rent & Return Movies: Rent available movies and return them, updating rental status in real-time.
  • Rental History: Track past rentals and their status.
  • Security: Password hashing and input sanitization for user data protection.
  • Additional Features: Movie search, admin panel for movie management, and user account management.
